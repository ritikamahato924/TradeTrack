# 📈 TradeTrack (Zerodha Clone)

TradeTrack is a full-stack stock trading platform inspired by **Zerodha trading platform clone built** to simulate real-world stock trading UI and workflows. This project demonstrates frontend design, backend integration, and dashboard functionality.

---

## 🚀 Features

* 📊 Interactive Dashboard (Holdings, Positions, Orders)
* 📈 Stock Watchlist UI
* 💰 Funds Management Interface
* 🔐 User Authentication
* ⚡ Fast frontend using Vite + React
* 🔄 Backend API integration

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### Other

* Git & GitHub

---

## 📁 Project Structure

```
Zerodha Clone/

├── backend/      # Server & APIs
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/ritikamahato924/TradeTrack.git
cd Zerodha-Clone
```

### 2. Install dependencies

#### Frontend

```
cd frontend
npm install
npm run dev
```

#### Dashboard

```
cd dashboard
npm install
npm run dev
```

#### Backend

```
cd backend
npm install
npm start
```
